# DOCUMENTO 12 — ROADMAP DEL PROYECTO

## Fase 1: MVP Fundacional (Semanas 1-3)
### Sprint 1: Infraestructura Base
- Setup monorepo, CI/CD básico
- Base datos PostgreSQL/Supabase
- Sistema autenticación JWT
- Registro/login usuarios
- Estructura paneles base

### Sprint 2: Marketplace Básico
- CRUD productos
- Listado/filtrado productos
- Motor económico base
- Carrito/checkout básico
- Historial compras

### Sprint 3: Sistema Donaciones
- Motor donaciones
- Checkout con donación opcional
- Transparencia donaciones
- Feed actividad básico

## Fase 2: Monetización (Semanas 4-6)
### Sprint 4: Sistema Planes
- Implementar Free/Pro/Premium/Elite
- Restricciones por plan
- Upgrade/downgrade planes
- Dashboard vendedor

### Sprint 5: Sistema MLM
- Motor comisiones
- Tracking enlaces promoción
- Dashboard promotor
- Árbol 2 niveles

### Sprint 6: Afiliaciones
- Alta afiliadores
- Comisiones nivel 2
- Dashboard afiliador

## Fase 3: Engagement (Semanas 7-9)
### Sprint 7: Causas y Campañas
- CRUD causas
- Progreso campañas
- Panel ONG
- Feed causas

### Sprint 8: Sistema Sorteos
- CRUD premios
- Motor sorteos
- Acumulación entradas
- Historial sorteos

### Sprint 9: Sorteos Avanzados
- Sorteos automáticos
- Notificaciones ganadores
- Panel administración

## Fase 4: Admin y Escala (Semanas 10-12)
### Sprint 10: Panel Admin
- Dashboard completo
- Gestión usuarios
- Configuración global
- Reportes

### Sprint 11: Auditoría y Seguridad
- Sistema auditoría completo
- Detección fraude
- Logs centralizados
- Alertas

### Sprint 12: Optimización y Launch
- Performance tuning
- Testing E2E
- Documentación final
- Deployment producción

## Hitos Clave
1. **Semana 3**: MVP funcional
2. **Semana 6**: Monetización activa
3. **Semana 9**: Engagement completo
4. **Semana 12**: Launch producción
