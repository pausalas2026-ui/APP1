# DOCUMENTO 6 — CAPÍTULO 5: MÓDULOS CLAVE

## 5.1. Usuarios, Roles y Planes
- Registro, login, roles, permisos, planes
- Upgrades/downgrades, verificación, tokens
- Free NO puede crear productos/tienda/premios

## 5.2. Causas y Campañas
- Crear causas, campañas, destino fondos
- Causa activa > causa base (siempre)
- Toda donación con causa destino asignada

## 5.3. Productos y Marketplace
- NINGÚN producto sin aprobación admin
- Comisión total la define creador
- Productos en tiendas personales de promotores

## 5.4. Donaciones (Carrito Solidario)
- Campo donación default 1€ (editable)
- Donación = evento económico separado
- Promotor puede donar parte de su comisión

## 5.5. Sorteos y Premios
- Todo sorteo requiere aprobación admin
- Puede requerir donación para participar
- Lógica de ganador transparente

## 5.6. Comisiones y MLM
- Nivel 1 (promotor), Nivel 2 (afiliador)
- NO existe nivel 3
- Valores históricos NO se modifican

## 5.7. Planes (Free/Pro/Premium/Elite)
- Elite = 100% multiplicador siempre
- Free nunca puede crear productos/premios/tienda

## 5.8. Tableros / Paneles
- Dashboard por rol
- Datos del backend, nunca frontend

## 5.9. Feed Inteligente
- Free = menor prioridad
- Elite = máxima visibilidad

## 5.10. Auditoría y Logs
- Registros inmutables
- Nunca modificar logs
